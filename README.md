# Movie Cards UI

## 📌 Description

A simple movie-themed webpage built using HTML and CSS. The project displays a collection of movie cards with interactive hover effects and smooth visual transitions. 
This project is focused on UI design and does not include functional buttons, navigation, or backend integration.

## ✨ Features

* Movie card layout
* Interactive hover animations
* Smooth scaling and shadow effects

## 🛠️ Technologies Used

* HTML5
* CSS3

## ⚙️ How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/Aashu-Goswami/movie-cards-UI.git
   ```

2. Launch `index.html`.

## 🧠 Concepts Practiced

* HTML page structure
* CSS Flexbox/Grid layouts
* Hover effects and transitions
* Card-based UI design
* Basic responsive design principles
