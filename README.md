# Movie Cards UI

# Description
This project is a simple static movie details page built using HTML and CSS.  
It demonstrates CSS hover effects on movie cards where cards expand slightly and shadows become darker when hovered.
The purpose of this project is to practice UI design and CSS transitions.

# Features
- Responsive movie card layout
- Hover animations on cards
- Smooth scaling effect on hover
- Shadow enhancement when hovering
- Clean and simple UI design

# Technologies Used
- HTML5
- CSS3
